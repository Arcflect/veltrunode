# 行動規範 (Code of Conduct)

## 私たちのコミットメント

私たちは、すべての参加者にとって安全で、生産的で、歓迎される環境を作ることを約束します。年齢、身体サイズ、障害、民族、ジェンダー特性、性自認および表現、経験レベル、教育、社会的地位、国籍、容姿、人種、宗教、または性的指向に関係なく、オープンでフレンドリーなコミュニティの維持に努めます。

## 期待される行動

- **敬意を持った対話**: コミュニティ内でのすべてのコミュニケーションにおいて、互いに敬意を払い、建設的な発言を心がけてください。
- **技術的な議論への集中**: 批判を行う際は個人を対象にするのではなく、技術的な成果物や提案そのものに向けてください。
- **ハラスメントと差別の禁止**: いかなる形態の嫌がらせ、差別的・屈辱的な言動も許容されません。
- **プライバシーの保護**: 他者の同意なく、連絡先や個人情報などの非公開情報を公開しないでください。

## メンテナーの責任

プロジェクトのメンテナーは、本行動規範の基準を明確にし、これを遵守する責任があります。本行動規範に反する行為、またはコミュニティの健全な環境を脅かす発言やコンテンツに対して、必要に応じて削除、修正、または参加停止などの措置を講じることがあります。

## 違反の報告

本行動規範に違反する行為を目撃した、または自身が受けた場合は、プロジェクトのメンテナーまでプライベートにご連絡ください。すべての報告は真摯に受け止められ、適切かつ機密に調査・処理されます。
